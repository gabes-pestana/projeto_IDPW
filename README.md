
# Projeto IDPW
Projeto de desenvolvimento web do primeiro semestre

# 1.Introdução
Esse trabalho consiste no desenvolvimento de uma página web para uma plataforma de músicas. Aplicando tenologias como HTML e CSS.

# 2.Objetivos
O trabalho cumpre como objetivos o uso de responsividade, tags semânticas para a estrutura da página, presença de formulários, listas, imagens e conteúdo multimídia. Além do uso de links e bookmarks. Também atendendo a alguns critérios de acessibilidade.

# 3.Tecnologias utilizadas
Para o desenvolvimento da página foi utilizado HTML e CSS.

# 4.Estrutura do projeto
A página principal do projeto está no arquivo "index.html". Na pasta "links", estão armazenados os links auxiliares que contém as páginas que possuem as demais interfaces do projeto. Na pasta "img" estão contidas as imagens utilizadas, e na pasta "music", a música utilizada para a tag audio. E a estilização do projeto está localizada no arquivo style, que possui a estilização de todas as páginas, com todo o conteúdo organizado por comentários orientando onde cada parte pertence.

# 5.Funcionalidades implementadas
1. Vlibras
Permite que usuários que necessitam da tecnologia possam usar o site.

2. Player de áudio
Permite ouvir músicas existentes na página.
